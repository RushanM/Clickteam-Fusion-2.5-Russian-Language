Основное меню (Main Menu)

Получить значение элемента (Get Item Value)

Получает значение, соответствующее указанному элементу как число. Если это возможно число будет целым, если нет числом с плавающей запятой. Первый параметр название группы, второй название элемента и последний параметр значение по умолчанию.

В случае если элемент не существует, используется значение по умолчанию. Значение по умолчанию всегда возвращается как плавающее.

Получить строку элемента (Get Item String)

Получает значение, соответствующее указанному элементу как строку. Первый параметр название группы, второй название элемента и последним параметр является значением по умолчанию. В случае если элемент не существует, используется значение по умолчанию.

Строка не может быть возвращена точно таким же образом, как отмечено в файле. Например, управляющие символы будут заменены символами, которые они представляют, и кавычки могут быть удалены.

Получить позицию X (Get X Position)

Первый и второй параметры определяют группу и элемент для чтения соответственно. Это выражение применяется к элементам, отформатированным в виде 'number1, number2'. Это выражение возвращается, как целое число, 'number1'. Если оно не существует, используется значение по умолчанию 0.

Получить позицию Y (Get Y Position)

Первый и второй параметры определяют группу и элемент для чтения соответственно. Это выражение применяется к элементам, отформатированным в виде 'number1, number2'. Это выражение возвращается, как целое число, 'number2'. Если оно не существует, используется значение по умолчанию 0.

Получить INI как строку (Get Ini as String)

Это действие получает в виде строки текста, которая будет сохранена в файл командой 'Сохранить (Save)'. Однако шифрование и сжатие не применяется.

Получить INI как CSV строку (Get Ini as CSV String)

Это действие получает в виде строки текста, которая будет сохранена в файл командой 'Импорт/Экспорт CSV (Import/Export as CSV)'.

Список групп и элементов (List Groups and Items)

Получить энное название группы (Get N-th Group Name)

Это выражение содержит список всех названий групп в объекте в алфавитном порядке. Параметр который возвращает номер группы. Первая группа 0 или 1 в зависимости от настроек в панели свойств.

Используйте выражение 'Получить количество групп (Get Group Count)', чтобы узнать сколько есть групп. Любое значение диапазона будет просто возвращать пустую строку.

Получить энное название элемента (Get N-th Item name)

Это выражение содержит список всех названий элементов в объекте в алфавитном порядке. Первый параметр указывает группу для просмотра и второй параметр какое число элемента должно быть возвращено. Первым значением элемента является либо 0, либо 1, в зависимости от настроек в панели свойств.

Используйте выражение 'Получить количество элементов (Get Item Count)', чтобы узнать сколько элементов есть в указанной группе. Любое значение диапазона будет просто возвращать пустую строку.

Получить энную строку элемента (Get N-th Item String)

Это выражение даёт значение связанное с номером элемента в списке, упорядоченном выражением 'Получить энное название элемента (Get N-th Item name)' (Которые отсортированы по названию ключа в алфавитном порядке).

Первый параметр указывает группу для просмотра и второй параметр какое число элемента должно быть возвращено. Первым значением элемента является либо 0, либо 1, в зависимости от настроек в панели свойств.

Получить энное значения элемента (Get N-th Item Value)

Это выражение даёт значение связанное с номером элемента в списке, упорядоченном выражением 'Получить энное название элемента (Get N-th Item name)' (которые отсортированы по названию ключа в алфавитном порядке). Значение преобразованное в число (если это возможно число будет целым, если нет с плавающей запятой).

Первый параметр указывает группу для просмотра и второй параметр какое число элемента должно быть возвращено. Первым значением элемента является либо 0, либо 1, в зависимости от настроек в панели свойств.

Получить энное отсортированное название элемента (Get N-th Sorted Item Name)

Это действие как 'Получить энное название элемента (Get N-th Item name)', но упорядоченные в алфавитном порядке по значениям, а не по названиям элементов.

Первый параметр указывает группу для просмотра и второй параметр какое число элемента должно быть возвращено. Первым значением элемента является либо 0, либо 1, в зависимости от настроек в панели свойств.

Используйте выражение 'Получить количество элементов (Get Item Count)', чтобы узнать сколько элементов есть в указанной группе. Любое значение диапазона будет просто возвращать пустую строку.

Получить энное отсортированное значение элемента (Get N-th Sorted Item Value)

Это действие как 'Получить энное значения элемента (Get N-th Item Value)', но упорядоченные в алфавитном порядке по значениям, а не по названиям элементов. Если это возможно значение возвращается как целое число, если нет с плавающей запятой.

Первый параметр указывает группу для просмотра в и второй параметр какое число элемента должно быть возвращено. Первым значением элемента является либо 0, либо 1, в зависимости от настроек в панели свойств.

Используйте выражение 'Получить количество элементов (Get Item Count)', чтобы узнать сколько элементов есть в указанной группе. Любое значение диапазона будет просто возвращать пустую строку.

Получить энный элемент используемый в любой группе (Get N-th Item used in any group)

Это выражение позволяет всем элементам обрабатываться независимо от группы. Возвращает название элементов. Элементы не повторяются.

Например, файл Ini:

Example Ini 23

[group]
Bob = 1
Ted = 2
[group 2]
Bob = 12
John = 22

Тогда список будет: 'Bob', 'Ted', 'John'.

Выражение 'Получить количество уникальных названий элементов (Get number of unique item names)' может быть использовано чтобы выяснить, сколько названий элементов есть. Это выражение соблюдает параметр '1 начальный индекс (1 based index)'. Пустая строка будет возвращена из диапазона.

Подсчёт и вычисления (Counting and Calculations)

Получить количество групп (Get Group Count)

Возвращает количество групп в объекте. Это включает в себя пустые группы если включено 'Разрешить пустые группы (Allow empty groups)'.

Получить количество элементов (Get Item Count)

Возвращает количество элементов в указанной группе.

Получить общее количество элементов (Get Total Item Count)

Возвращает общее количество элементов в любой группе (включая элементы с одинаковыми именами, но в разных группах).

Получить количество уникальных названий элементов (Get number of unique item names)

Возвращает количество уникальных названий элементов в объекте. Полезное с выражением 'Получить энный элемент используемый в любой группе (Get N-th Item used in any group)'.

Выполнить вычисления (Perform Calculations)

Это выражение является немного более сложным, чем другие.

Идея выражения, чтобы получить данные из объекта путем выполнения вычисления в каждой группе, а затем выполнить вычисления, чтобы объединить все эти цифры. Например, мы могли бы найти среднее значение каждой группы, а затем взять произведение всех этих средних значений.

Это лучше всего показано на примерах.

Например, предположим что у Вас есть следующий файл (записи Arsenal против различных команд со времен войны):

Example Ini 24

[Bradford City]
Win=10
Draw=5
Lose=9

[Ajax]
Win=1
Draw=0
Lose=3

[Barcelona]
Win=0
Draw=1
Lose=4

[Wimbledon]
Win=13
Draw=8
Lost=7

Предположим, мы хотим знать, сколько матчей в общей сложности было против этих команд. Это очень просто с этим выражением. Установка второго параметра на «+» приведет к тому каждая из групп суммируется. Мы получим следующие значения, связанные с каждой группой: Bradford City, 24; Ajax: 4; Barcelona: 5; Wimbledon: 28. Если необходимо чтобы эти значения добавлялись, для первого параметра мы выбираем «+», а затем выражение возвращает: 24+4+5+28 = 61.

Теперь мы хотим знать общее количество игр вничью(Draw) с каждой из этих команд. Чтобы сделать это для второго параметра мы просто написали «Draw». Мы получим следующие значения, связанные с каждой группой: Bradford City, 5; Ajax: 0; Barcelona: 1; Wimbledon: 8. В качестве первого параметра мы снова пишем '+', и выражение возвращает: 14.

Если Вам нужно количество игр против Ajax, Вы пишите «Ajax» для первого параметра и '+' для второго параметра. Выражение возвращает: 4.

Ниже приведены некоторые общие примеры:

  • 'Avg' в качестве первого параметра и 'Max' для второго параметра вернёт среднее максимальное значение в группе.
  • 'Max' в качестве первого параметра и 'n' для второго параметра вернёт размер самой большой группы (количество содержащихся элементов в группе).
  • 'iAvg' в качестве первого параметра и '+' для второго параметра вернёт среднее значение элемента в Ini++ объекте. (Это происходит потому, что он суммирует всё, а затем делит на количество элементов.)

Найти Мин. или Макс. группы (Find Max or Min Group)

Работа этого выражения очень похожа на 'Выполнить вычисления (Perform Calculations)'. Элементы в каждой группе объединены в соответствии со вторым параметром, которым может быть любым из операторов признанным верным для выражения 'Выполнить вычисления (Perform Calculations)'. Однако вместо того объединения этих значений группы вместе, это выражение возвращает название группы связанной либо с максимальным либо с минимальным значением. Поэтому первый параметр может быть только 'Max' или 'Min'.

В примере из секции 'Выполнить вычисления (Perform Calculations)', мы могли бы использовать это выражение с первым параметром 'Max' и вторым параметром 'Win', чтобы узнать название команды которую Arsenal победил большее количество раз (Wimbledon).

Дополнительные функции (Extra Functions)

Хеш строки (Hash String)

Возвращает хеш указанной строки.

Управляющая строка (Escape String)

Это выражение меняет '\' с '\\', кавычки с '\q', вкладки с '\t' и новую строку с '\n'. Новые линии любого символа новой строки в настройках объекта.

Не управляющая строка (Unescape String)

Это выражение меняет '\\' с '\', '\q' с кавычками, '\t' с вкладками, '\n' с новой строкой и '\ ' с пробелами.
