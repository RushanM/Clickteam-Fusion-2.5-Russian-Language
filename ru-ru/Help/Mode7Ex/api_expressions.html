Mode 7 Ex Ссылка API: выражения


  • Размер(Size)
    • Получить ширину(Get Width) — возвращает ширину Mode 7 Ex объектов на экране
    • Получить высоту(Get Height) — возвращает высоту Mode 7 Ex объектов на экране

  • Изображения(Images)
    • Основное изображение карты(Main Image Map)
      • Получить ширину изображения(Get Image Width) — возвращает текущую ширину изображения загруженного в основную поверхность
      • Получить высоту изображения(Get Image Height) — возвращает текущую высоту изображения загруженного в основную поверхность
      • Получить номер изображения(Get Image Number) — возвращает номер идентификатора текущего загруженного изображения в основную поверхность, -1 для внешнего изображения
      • Получить название изображения(Get Image Name) — возвращает название внутреннего изображения или название внешнего файла в зависимости от контекста

    • Вторичное изображение карты(Secondary Image Map)
      • Получить ширину изображения(Get Image Width) — возвращает текущую ширину изображения загружаются во вторичную поверхность
      • Получить высоту изображения(Get Image Height) — возвращает текущую высоту изображения загружаются во вторичную поверхность
      • Получить номер изображения(Get Image Number) — возвращает номер идентификатора текущего загруженного изображения во вторичную поверхность, -1 для внешнего изображения, -2 если отсутствует
      • Получить название изображения(Get Image Name) — возвращает название внутреннего изображения или название внешнего файла в зависимости от контекста

    • Изображение карты высоты(Image Height Map)
      • Получить ширину изображения(Get Image Width) - возвращает текущую ширину изображения загруженного в качестве карты высоты
      • Получить высоту изображения(Get Image Height) - возвращает текущую высоту изображения загруженного в качестве карты высоты
      • Получить номер изображения(Get Image Number) - возвращает номер идентификатора текущего загруженного изображения в качестве карты высоты, -1 для внешнего изображения, -2 если отсутствует
      • Получить название изображения(Get Image Name) - возвращает название внутреннего изображения или название внешнего файла в зависимости от контекста
      • Получить высоту по(Get Height At) - возвращает высоту Mode 7 Ex в единицах уровня в конкретной точки х, у

  • Настройки обработки(Render Options) - Для более подробного объяснения настроек обработки, обратитесь к ссылке на ‘Действия’
    • Разрешение отображения(Display Resolution)
      • Получить разрешение по горизонтали(Get Horizontal Resolution)
      • Получить разрешение по вертикали(Get Vertical Resolution)
      • Получить режим разрешения(Get Resolution Mode) - возвращает номер 0-3
      • Получить параметры режима разрешения(Get Resolution Mode Parameter) - возвращает число 0-255

    • Режим прозрачности(Transparency Mode)
      • Получить включён (1: вкл.)(Get Enabled (1=on) - возвращает 1 если режим прозрачности включён, 0 если выключен

    • Уровень и высота(Thickness and Height)
      • Получить уровень(Get Thickness) - возвращает текущий уровень/максимальную высоту
      • Получить режим уровня(Get Thickness Mode) - возвращает число 0-2

    • Интерполяция(Interpolation)
      • Режим интерполяции изображение (1: вкл.)(Image Interpolation Mode (1=on)) - возвращает 1 если интерполяция изображения включена, 0 если выключена
      • Режим перепада интерпаляции(1: вкл.)(Elevation Interpolation Mode (1=on)) - возвращает 1, если перепад высоты/интерполяция карты высоты включена, 0 если выключена

    • (Mip-Mapping)
      • Получить режим Mip-Mapping изображения (1: вкл.)(Get Image Mip-Mapping Mode (1=on)) - возвращает 1 если mip-mapping is основной поверхности включён, 0 если выключен
      • Получить режим Mip-Mapping вторичного изображениея (1: вкл.)(Get Secondary Image Mip-Mapping Mode (1=on)) - возвращает 1 если mip-mapping is вторичной поверхности включён, 0 если выключен
      • Получить режим Mip-Mapping изображения карты высоты (1: вкл.)(Get Image Height Map Mip-Mapping Mode (1=o)n) - возвращает 1 если mip-mapping is карты высоты включён, 0 если выключен

    • Режим обрезки(Wrap Mode)
      • Получить горизонталь (1: вкл.)(Get Horizontal (1=on)) - возвращает 1 если горизонтальная обрезка включена, 0 если выключена
      • Получить вертикаль (1: вкл.)(Get Vertical (1=on)) - возвращает 1 если вертикальная обрезка включена, 0 если выключена

    • Туман(Fog)
      • Включён режим (1: вкл.)(Enabled Mode (1=on)) - возвращает 1 если туман включён, 0 если выключен
      • Цвет(Color)
        • Получить красный(Get Red) - возвращает красный компонент цвета тумана
        • Получить зелёный(Get Green) - возвращает зелёный компонент цвета тумана
        • Получить синий(Get Blue) - возвращает синий компонент цвета тумана
      • Получить сумму(Get Amount) - возвращает сумму тумана

    • Получить точку изчезновения(Get Vanishing Point) - не используется
    • Получить коэффицент масштабирования(Get Zoom factor) - возвращает коэффициент масштабирования в процентах

  • Вращение(Rotation)
    • Получить угол наклона (X):(Get Tilt Angle (X)) - не используется
    • Получить угол поворота (Y)(Get Rotation Angle (Y)) - возвращает текущий угол поворота камеры в градусах
    • Получить крен (Z)(Get Roll (Z)) - не используется

  • Начало(Origin)
    • Получить X(Get X) - возвращает текущую горизонтальную позицию камеры
    • Получить Y(Get Y) - возвращает текущую вертикальную позицию камеры
    • Получить Z (перепад)(Get Z (Elevation)) - возвращает текущее расстояние до поверхности

  • Вторичный слой(Secondary Layer)
    • Получить позицию X горячей точки — возвращает текущую горизонтальную позицию вторичной поверхности
    • Получить позицию Y горячей точки — возвращает текущую вертикальную позицию вторичной поверхности
    • Получить угол поворота по Y — возвращает текущий вторичный угол в градусах

  • Маппирование спрайта
    • Получить X на экране из X,Y(Get on-screen X from X,Y) - возвращает отображение на экране X координаты спрайта по X, Y на поверхности Mode 7 Ex
    • Получить Y на экране из X,Y(Get on-screen Y from X,Y) - возвращает отображение на экране Y координаты спрайта по X, Y на поверхности Mode 7 Ex
    • Получить ширину на экране из X,Y,Ш,В(Get on-screen width from X,Y,W,H) - возвращает отображение ширины по X,Y на поверхности Mode 7 Ex если это 'Верно' ширина и высота это ШxВ
    • Получить высоту на экране из X,Y,Ш,В(Get on-screen height from X,Y,W,H) - возвращает отображение высоты по X,Y на поверхности Mode 7 Ex если это 'Верно' ширина и высота это ШxВ
    • Получить X на экране из X,Y,Z(Get on-screen X from X,Y,Z) - возвращает отображение на экране X координаты спрайта по X, Y с перепадом высоты Z на поверхности Mode 7 Ex
    • Получить Y на экране из X,Y,Z(Get on-screen Y from X,Y,Z) - возвращает отображение на экране Y координаты спрайта по X, Y с перепадом высоты Z на поверхности Mode 7 Ex
    • Получить угол на экране из X,Y,А(Get on-screen angle from X,Y,A) - возвращает отображение угла спрайта по X, Y в направление к углу А
    • Видимое X,Y,Ш,В (1: Видимое)(Visible at X,Y,W,H (1=visible)) - возвращает 1, если спрайт по X, Y с шириной и высотой ШxВ должен быть виден на экране, 0, если недолжен быть виден
    • Видимое X,Y,Z,Ш,В (1: Видимое)(Visible at X,Y,Z,W,H (1=visible)) - возвращает 1, если спрайт по X, Y с перепадом высоты Z, шириной Ш и высотой В должен быть виден на экране, 0, если недолжен быть виден

  • Перерисовка(Redraw)
    • Включить Автоперерисовку (1: вкл.)(Auto-Redraw Enabled (1=on)) - возвращает 1 если автоперерисовка включена, 0 если выключена
