Mode 7 ex Ссылка API: Действия

Действия в Mode 7 ex отображаются здесь, как они представлены в меню Mode 7 ex в Multimedia Fusion.

  • Размер(Size)
    • Установить ширину/Установить высоту(Set Width/Set Height) - устанавливает ширину и высоту объекта Mode 7 ex на экране

  • Изображения(Images) - Примечание: Номера изображений используют нулевой индекс, первое изображение 0, второе 1, третье 2, и т. д.
    • Основное изображение карты(Main Image Map)
      • Загрузить файл(Load File) - загружает новое изображение (bmp, pcx, или gif) из файла в основную поверхность
      • Загрузить изображение номер(Load Image number) - загружает внутреннее изображение в основную поверхность
      • Загрузить изображения с названием(Load Image name) - загружает внутреннее изображение в основную поверхность по названию (указано в Опциях настройки)

    • Вторичное изображение карты(Secondary Image Map)
      • Загрузить файл(Load File) - загружает новое изображение (bmp, pcx, или gif) из файла во вторичную поверхность
      • Загрузить изображение номер(Load Image number) - загружает внутреннее изображение во вторичную поверхность
      • Загрузить изображения с названием(Load Image name)- загружает внутреннее изображение во вторичную поверхность по названию (указано в Опциях настройки)
      • Выгрузить(Unload) - выгружает и сбрасывает текущее вторичное изображение

    • Изображение карты высоты(Image Height Map) - Примечание: вы должны указать уровень до того, как карта высоты будет использоваться
      • Загрузить файл(Load File) - загружает новое изображение (bmp, pcx, или gif) из файла как карту высоты
      • Загрузить изображение номер(Load Image number) - загружает внутреннее изображение как карту высоты
      • Загрузить изображения с названием(Load Image name) - загружает внутреннее изображение как карту высоты по названию (указано в Опциях настройки)
      • Выгрузить(Unload) - выгружает и сбрасывает текущую карту высоты

  • Настройки обработки(Render Options) - многие из настроек обработки также могут быть инициализированы из Опциий настройки
    • Разрешение отображения(Display Resolution) - разрешение снижает качество mode 7 ex обработки для более высокой скорости, разбив его на блоки
      • Установить размер разрешения(Set Resolution Size) - устанавливает горизонтальное и вертикальное разрешение в одном шаге, 1x1 это полная обработка
      • Установить режим разрешения(Set Resolution Mode) - устанавливает тип отображаемого разрешения, указанный как номер и дополнительный параметр
        • 0: Быстрый(quick) - рисует один пиксель для каждого обработанного пикселя
        • 1: Полный(full) - растягивает каждый обработанный пиксель , чтобы заполнить размер указанного разрешения
        • 2: Строки развёртки(scanlines) - создаёт строки горизонтальной развёртки, рисуя лишь каждую первую строку(используйте соотношение разрешения 2х2)
        • 3: % scanlines - те же строки горизонтальной развёртки, но со смешанными строками (использует параметр непрозрачности 0-255)
      • Установить разрешение по горизонтали(Set Horizontal Resolution)
      • Установить разрешение по вертикали(Set Vertical Resolution)

    • Режим прозрачности(Transparency Mode)
      • Enable/Disable - переключает режим прозрачности чёрного

    • Уровень и высота(Thickness and Height) - Если не загружена карта высоты, все изображение будет установлены на уровень. Уровень использует те же единицы что и перепад.
      • Установить уровень(Set Thickness) - определяет максимальный уровень Mode 7 ex поддержки вокселей. Если загружена карта высоты, любой чёрный/ 0,0,0 будет плоским, и белый/255,255,255 достигнет максимального уровня
      • Установить режим уровня(Set Thickness Mode) - контролирует как ‘скрытые’ пиксели будут рассчитаны и нарисованы
        • 0: Быстрый(fast) - скрытые пикселей игнорируются и могут быть видны через
        • 1: Заполнить(fill) - 'кусок' создаётся там, где находятся скрытые пиксели
        • 2: Полный(full) - в этом режиме Mode 7 ex самый медленный, пытается вычислить как много скрытых пикселей, насколько это возможно

    • Интерполяция(Interpolation) - Примечание: интерполяция значительно замедлит Mode 7 ex
      • Включить/Выключить интерполяцию изображения(Enable/Disable Image Interpolation) - переключение интерполяции/сглаживания на основных и вторичных поверхностях
      • Включить/Выключить перепад интерполяции(Enable/Disable Elevation Interpolation) - переключение интерполяции/сглаживания на карте высоты

    • (Mip-Mapping) - mip-mapping выполняет большую часть вычислений при загрузке изображения, что делает его довольно быстрой функцией
      • Включить/Выключить Mip-Mapping изображения(Enable/Disable Image Mip-Mapping) - переключает mip-mapping на основной поверхности
      • Установить прозрачный цвет Mip-Map(Set Mip-Map Transparent Color) - указать цвет для использования прозрачности в основной поверхности mip-mapping
      • Очистить прозрачный цвет Mip-Map(Clear Mip-Map Transparent Color) - сбрасывает цвет прозрачности по умолчанию
      • Включить/Выключить Mip-Mapping вторичного изображения(Enable/Disable Secondary Image Mip-Mapping) - переключает mip-mapping на вторичном изображении
      • Включить/Выключить Mip-Mapping изображения карты высоты(Enable/Disable Image Height Map Mip-Mapping) - переключает mip-mapping на карте высоты (может создать непредсказуемые результаты)

    • Режим обрезки(Wrap Mode)
      • Включить/Выключить горизонтальную обрезку(Enable/Disable Horizontal Wrap) - включает/выключает горизонтальную обрезку основного изображения поверхности
      • Включить/Выключить вертикальную обрезку(Enable/Disable Vertical Wrap) - включает/выключает вертикальную обрезку основного изображения поверхности

    • Туман(Fog)
      • Включить/Выключить(Enable/Disable) - включает/выключает туман
      • Цвет(Color) - Mode 7 ex предоставляет несколько методов корректировки цвета тумана
        • Выбрать цвет(Choose Color) - стандартный выбор цвета Multimedia Fusion
        • Установить цвет(Set Color) - красный, зелёный и синий компоненты как номера
        • Установить красный(Set Red) - корректировать только красный компонент
        • Установить зелёный(Set Green) - корректировать только зелёный компонент
        • Установить синий(Set Blue) - корректировать только синий компонент
      • Установить сумму(Set Amount) - сумма тумана на экране, 0 тумана нет, 50000 покрывает большую часть или весь экран
    • Установить точку изчезновения(Set Vanishing Point) - не используется
    • Установить коэффициент масштабирования(Set Zoom factor) - процент (100 по умолчанию), влияет на координаты и отображение изображения

  • Вращение(Rotation)
    • Угол наклона (X)(Tilt Angle (X)) - не используется
    • Угол поворота (Y)(Rotation Angle (Y)) - установить угол в градусах, куда камера смотрит
    • Крен (Z)(Roll (Z)) - не используется
    • Корректировка угла наклона (X)(Adjust Tilt Angle (X)) - не используется
    • Корректировка угла вращения (Y)(Adjust Rotation Angle (Y)) - увеличить угол в градусах, куда камера смотрит(или используйте отрицательное число для уменьшения)
    • Корректировка крена угла (Y)(Adjust Roll (Z)) - не используется

  • Установить начало(Set Origin) - начало это местоположение камеры/зрителя на поверхности Mode 7 ex
    • X - установить горизонтальную позицию
    • Y - установить вертикальную позицию
    • Z (перепад)(Z (Elevation)) - установить расстояние от поверхности, отрицательный перепад высоты будет ‘отражать’ изображение Mode 7 ex для потолков и облаков
    • Переместить X(Slide X) - увеличить горизонтальную позицию (или используйте отрицательное число для уменьшения)
    • Переместить Z(Slide Y) - увеличить вертикальную позицию (или используйте отрицательное число для уменьшения)
    • Переместить Z(Slide Z) - увеличить дистанцию от поверхности (или используйте отрицательное число для уменьшения)
    • Идти вперёд(Walk Forward) - движение игрока ‘вперёд’ на основе угла поворота (Y)
    • Идти назад(Walk Back) - движение игрока ‘назад’ на основе угла поворота (Y)
    • Идти влево(приставные шаги)(Walk Left (Strafe)) - перемещение игрока ‘влево’ на основе угла поворота (Y)
    • Идти вправо(приставные шаги)(Walk Right (Strafe)) - перемещение игрока ‘вправо’ на основе угла поворота (Y)

  • Вторичный слой(Secondary Layer) - вторичная поверхность может иметь независимое от основной поверхности положение и угол
    • Установить начальный X(Set X Origin) - установить горизонтальную позицию
    • Установить начальный Y(Set Y Origin) - установить вертикальную позицию
    • Переместить начальный X(Slide X Origin) - увеличить горизонтальную позицию (или используйте отрицательное число для уменьшения)
    • Переместить начальный Y(Slide Y Origin) - увеличить вертикальную позицию (или используйте отрицательное число для уменьшения)
    • Установить угол поворота (Y)(Set Rotation Angle (Y)) - установить угол в градусах
    • Корректировка угла вращения (Y)(Adjust Rotation Angle (Y)) - увеличить угол в градусах(или используйте отрицательное число для уменьшения)

  • Перерисовка(Redraw) - в основном вам не нужно трогать эти параметры
    • Включить/Выключить Автоперерисовку(Enable/Disable Auto Redraw) - включает/выключает автоматическую перерисовку экрана. Если эта опция выключена Вам нужно будет использовать принудительную перерисовку
    • Принудительная перерисовка(Force Redraw) - принудительно Mode 7 ex для перерисовки экрана следующего кадра
