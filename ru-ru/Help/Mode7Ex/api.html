Режим 7: API

Раздел про API разбит на три различных раздела. Справка подразумевает базовое общее понимание использования Multimedia Fusion.

» Окно настройки и настройки объекта
Если дважды нажав на объект Режима 7 появится диалоговое окно параметров, что позволяет изменять внутренние изображения и быстро настроить многие из начальных установок Mode 7 ех.

» Действия
Контроль действий расширения позволяет перемещать, вращать и изменять отображение в режиме реального времени.

» Выражения
Выражения позволяют получать информацию о текущем состоянии объекта Режима 7.

Где условия?

Расширение Режима 7 сделано таким образом, что все функциональные возможности могут быть доступны через действия и выражения. Вы можете ‘создать’ свои собственные условия, используя 'Сравнение двух общих значений' и одно из выражений объекта Mode 7 Ex.
