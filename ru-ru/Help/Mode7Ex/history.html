Mode 7 Ex История версии

История версия распространяется только на Mode 7 Ex версии 1.0 и более поздние. Обратитесь к Что нового в Mode 7 Ex странице для получения сведений о добавленных функциях в Mode 7 Ex версии 1.0

 

Версия v1.01

  • Добавлены дополнительные проверки внутренних поверхностей - предотвращает сбои на компьютерах некоторых пользователей
  • Исправлена ошибка Получить карту/GetMap XZ (X, Y, Z) выражение вызывающее неверные расчёты для Mode 7 Ex поверхностей не находящихся в точке х=0.
  • Добавлена новая функция ПолучитьКартуВидимых/GetMapVisibleZ(X,Y,Z,W,H) для вычисления видимости ‘летающих’ спрайтов
