Mode 7 Ex Примеры

Это примеры объекта Mode 7 Ex. Все примеры предназначены для Multimedia Fusion версии не меньше 1.5, хотя само по себе расширение совместимо с более ранней версией Multimedia Fusion.

Пузыри/Bubbles - начальный

Наслоение нескольких объектов Mode 7 Ex может создать разные интересные эффекты. Этот пример демонстрирует лишь некоторые виды эффектов, которые можно создать.

Без управления.

Шахматная доска/Checkerboard - начальный

Mode 7 Ex не ограничивается небольшими масштабами, эта демонстрация работает в окне 640х480 с использованием режима разрешения.

Без управления.

Бесконечная дорога/Endless Road - начальный

Пример бесконечной дороги использует функцию горизонтальной обрезки, чтобы создать единый путь, который никогда не заканчивается. Кроме того синусоида используется для создания эффекта камеры 'качания автомобиля' .

Управление: Стрелки курсора для движения

Демонстрация логотипа/Logo Demo - начальный

Mode 7 Ex может создать необходимые эффекты текста и логотипа с помощью нескольких слоёв. Этот пример касается только эффектов поверхности в Mode 7 Ex.

Управление: Пробел переключает стиль логотипа

Демонстрация начала миссии/Mission Demo - начальный

Вы можете оживить ваши 2-D игры используя введение перед игрой на каждом уровне, используя режим 7 ех с поддержкой уровня высоты.

Туманная долина/Misty Valley - начальный

Летать через бесконечную местность.

Управление: Стрелки курсора для движения

Демонстрация Квазимодо/Quasi Demo - начальный

Порт оригинального демонстрационного примера mode 7.

Без управления.

Демонстрация текста/Text Demo - начальный

Классическая прокрутка 3D текст как в Star Wars(фильм: Звёздные войны).

Управление: Пробел ускоряет прокрутку

Остров/Island - средний

Простая демонстрация вокселей и отображение высоты. Используются два слоя облаков для показа большей глубины облаков. Кроме того, камера удерживается над землёй с помощью сопоставления высоты.

Keys: Стрелки курсора для движения, 1,2,3 для переключения режима уровня между: Быстрый/fast, Заполнить/fill и Полный/full

Демонстрация снега/Snow Demo - средний (требуется объекты быстрый цикл/fast loop и Слои/layers )

Использует отображается спрайта мяча и движения по траектории. Использует объект слои/layers на слое спрайтов для корректного отображения на экране. Также вычисляет количество кадров в секунду.

Управление: Стрелки курсора для движения, page up/down чтобы изменить высоту, t чтобы переключить рельеф местности.

Калькулятор скорости/Speed Calculator - средний

В этом примере предпринимается попытка создать режим идеального разрешения с использованием или без интерполяции изображения на основании количества кадров в секунду.

Без управления.

Отображение спрайтов/Sprite Mapping - средний (требуется объект быстрый цикл/fast loop)

Демонстрирует способ отображения нескольких спрайтов.

Управление: Стрелки курсора для движения

Шутер от первого лица/First Person Shooter - расширенный

Полноценная игра, в которой вы можете ходить вокруг арены Mode 7 Ex , стрелять и поражать врагов. Демонстрирует отображения нескольких спрайтов и столкновения спрайта со спрайтом.

Управление: Стрелки курсора для движения, Пробел для выстрела

Гоночный трек 3Д/Racing Line 3-D - расширенный

Преобразование Click and Create игры 'Racing Line 2'(поставляется вместе с Click and Create и Multimedia Fusion). В дополнение к гонкам по трассе, была создана карта высоты, чтобы дать треку реальную глубину. Моделирование гравитации используется для создания прыжков!

Управление: Стрелки курсора для движения
